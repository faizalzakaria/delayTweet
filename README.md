## DelayTweet

Stream tweet with a delay of N. The N can be configured.

## Architecture



## Code Structures

- app: All logics are in 
- config: All non app code is here, mostly around boot up, env, routes etc.

## Pre-requisites

- Node version 8.7.0 (Only tested with this)
- `yarn install`

## How to run

#### For server

- `npm run dev`

#### For test client

- `node client/test`
